Because this project evolves day by day, the best way to see the growth and specific modifications is through the Git History. You can track how a specific file has changed over time by following these steps:

1.) Navigate to any HTML file in this repository.

2. ) Click on the "History" button located at the top right of the file view.

3. )Browse through the Commits to see exactly what code was added, modified, or removed on any given day.
